## 搭建Webpack环境

* 全局安装 （npm install webpack webpack-cli -g） 卸载 （npm uninstall webpack webpack-cli -g）使用（wepack index.js）
* 项目中安装，在项目中（npm install webpack webpack-cli --save-dev ）使用（npx webpack index.js）
* npx webpack -v  查看项目中webpack版本
